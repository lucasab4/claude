# Minimal Portfolio

A sleek, minimalist personal portfolio website built with Next.js and Tailwind CSS.

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

This project is configured for easy deployment on Vercel.

## Customization

Modify the content in `src/app/page.tsx` to personalize your portfolio.
